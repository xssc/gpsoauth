# Gpsoauth

Direct ruby port of the gpsoauth python lib: https://github.com/simon-weber/gpsoauth

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gpsoauth-rb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gpsoauth-rb
    
## Usage

    require 'gpsoauth'
